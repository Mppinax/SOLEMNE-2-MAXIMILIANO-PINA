## INTRODUCCION AL PROYECTO

Este proyecto se enmarca dentro del Diseño generativo y el Arte cinético, entendiendo el sistema como una exploración personal de estas corrientes.

El trabajo está influenciado por intereses estéticos centrados en el alto contraste, el movimiento constante y formas aparentemente erráticas, que aunque parecen caóticas, responden a reglas internas controladas. El sistema busca representar la complejidad del universo y el espacio, donde el aparente desorden siempre está regulado por un orden subyacente.

Como referencia, se utilizaron las obras del **moodboard** relacionadas con el arte cinético y el diseño generativo, especialmente aquellas de carácter “espacial”, donde el movimiento y la vibración visual surgen desde estructuras formales precisas.

A partir de esto, el proyecto se define como “Supernova”, entendida como una explosión estelar de gran magnitud que libera una enorme cantidad de energía y luz en un corto periodo de tiempo.
